About Sam's Donut Maker Game

This is a game that allows the user to create donuts by clicking on the "Donut Clicker" button. The user can then use the donuts created to purchase one of two options: the auto clicker or the donut multiplier. The donut multiplier increases the number of donuts created by 20% with each multiplier purchased. The auto clicker automatically creates the number of auto clickers purchased multiplied by the value of each donut click for the user each second. The goal of the game is to get the number of donuts as high as you can. Have fun!
